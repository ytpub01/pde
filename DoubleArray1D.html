DoubleArray1D Samples

Class DoubleArray1D Provides a "light weight" one dimensional array structure with initialization capabilities, array operations, and optional bounds checking.

The default beginning index value is 0 : (C convention)

The copy constructor creates a duplicate instance. Deleting the copy will not delete the data of the original.

Constructors

DoubleArray1D() Creates a null class instance.
DoubleArray1D(long m) Creates a vector of length m.
DoubleArray1D(double* d, long m) Creates a vector of length m that "wraps" the data specifed by d. d is a pointer to a double array of size m. Operations on the vector will result in operations on the data d.
DoubleArray1D(const DoubleArray1D& V) Creates a duplicate of the vector V.
Initialization

initialize(long m) Initializes an instance to a vector of size m.
initialize(double* d, long m) Initializes an instance to a vector of length m that "wraps" the data specified by d. d is a pointer to a double array of size m. Operations on the vector will result in operations on the data d. argument.




Access/Sizing

double operator()(long i) Returns the element at location i. This operator may be invoked on either side of an assignment statement. The default indexing starts at 0.
long getSize() Returns the size of the vector.
long getIndexBegin() Returns the beginning index value.
long getIndexEnd() Returns the ending index value.
setIndexBegin(long k) Sets the beginning access index to k.
