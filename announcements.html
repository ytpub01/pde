Using jsMath: As of Assignment 7, I've started using jsMath to create the mathematics symbols for the class web pages. If you have trouble viewing the pages, then click on the jsMath box in the lower right corner of the browser to bring up a menu that will allow you to fiddle with the options. Also, you may want to use the print feature of the jsMath menu to print copies of the assignment, rather than using the standard print comand from your browser. Another way to open up the jsMath menu is to hold down the Alt key and then click on any displayed mathematics.

One of the advantages of using jsMath is that if you change the default font size used in your browser then the equations will also be changed, e.g. larger equations will occur with larger font sizes.
