Computational Problem

When developing computational methods for time-dependent PDE's it is very useful to be able to view plots of the solutions at different times. The purpose of the computational part of the first assignment is to have you run a sample code and either view plots of the solution as it is computed (for those who decide to use the CAMgraphics classes) or view plots created from data files after it has run. If you are using Visual C++, documentation to help you set up a Win32 Console Projects to run the assignments is available from the UsingVCPP7 web page.

For those who are not using the CAMgraphics classes

[1] Download the sample "main" routine Assign1.cpp and the required support files ExactPeriodicSolution.h, DoubleArray1D.h , and OutputProfile.h. Compile and run the code. Create plots of the output files, and turn in a plot of the solution corresponding to time t = 2.25.

For those who are using the CAMgraphics classes

[0] Download and unzip the CAMgraphics support files contained in CAMgraphics.zip. CAMgraphics.zip contains a subdirectory camgraph that
contains all of the required support files and a .lib file containing compiled executable.

[1] Create a Win32 Console Project (not a Console Application) with the settings described below. Download and insert into this project the sample "main" routine Assign1g.cpp and the required support files ExactPeriodicSolution.h, DoubleArray1D.h . Compile and run the program. As the program runs it will stop at the output times and a dialog will appear with a plot of the solution. To continue, hit the continue button. If you want hardcopy, select print or specify a file to contain the output. Turn in a plot of the solution corresponding to time t = 2.25.

Required Project Settings and Support Files To Use CAMgraphics:

A program that uses the CAMgraphics display requires project settings that are different than the default settings. Specifically, you use a Win32 Console Project with the following additional entries and settings

  • Specify the use of MFC in a DLL (How To V7)
  • Specify the use of the debug multi-threaded dll runtime library (How to V7)
  • Disable the use of pre-compiled headers (PCH's).(How To V7)
  • Add to your include path the location of the subdirectory camgraph containing the header files for CAMgraphics. (How To V7)
  • Add the startup file (from from subdirectory camgraph) CAMstartup.cpp to your project.
  • Add the library file (from from subdirectory camgraph) VC7Graphics.lib to your project. The .lib files will not show up in the file chooser dialog unless you specify file types *.* (arrgh!).
  • Add the resource (from from subdirectory camgraph) DisplayDialog.rc to your project.

Notes : The graphics display dialog may appear behind other windows, to make it visible, select the MSVC display icon from the bottom menu bar.
